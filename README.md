# Zaku-cs
C# .NET6 MVC version<br>

## Create project
```
dotnet new mvc \
    -f net6.0 \
    -lang "C#" \
    -n Zaku
```

## Run
```
dotnet run
```

## Build
```
dotnet build
```

## Nuget
```
dotnet add package CryptoExchange.Net --version 5.2.4
dotnet add package Bybit.Net --version 1.2.1
dotnet add package CCXT.NET --version 1.4.13
```
